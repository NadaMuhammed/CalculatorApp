# Calculator App
A simple calculator that does the main functionalities such as adding, subtracting, multiplying, ..etc.

# Built  With
* Java
* Constraint Layout

# Features
* Add
* Subtract
* Multiply
* Divide
* Square Root
* Decimal Numbers
* Clear And Delete
